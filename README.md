## קוד Apps Script מעודכן
צריך להחליף את הקוד ב־Apps Script בקוד הזה, כדי שגם נתוני בקרת המסך יישמרו:

```javascript
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);

    sheet.appendRow([
      new Date(),
      data.studentName || "",
      data.studentClass || "",
      data.teacherName || "",
      data.score || "",
      data.percent || "",
      data.q1 || "",
      data.q2 || "",
      data.q3 || "",
      data.q4 || "",
      data.q5 || "",
      data.q6 || "",
      data.q7 || "",
      data.q8 || "",
      data.q9 || "",
      data.q10 || "",
      data.q11a || "",
      data.q11b || "",
      data.q11c || "",
      data.focusLossCount || 0,
      data.copyPasteCount || 0,
      data.fullscreenExitCount || 0,
      data.elapsedMinutes || "",
      data.securityStatus || "",
      data.securityEvents || ""
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({ status: "success" }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    return ContentService
      .createTextOutput(JSON.stringify({ status: "error", message: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```
